Hoe het werkt

Van aanmelding tot werkende AI-assistent in 48 uur. Geen technische kennis nodig.

①

Aanmelden

Vul het formulier in met je wensen. Welk pakket wil je? Gebruik je Telegram of liever iets anders? We vragen alleen wat we nodig hebben.

②

Wij bouwen

Wij regelen een veilige server, installeren OpenClaw, configureren alles, en testen of het werkt. Jij hoeft niets te doen.

③

Jij praat

Je krijgt toegang tot je AI-assistent via Telegram (of een ander kanaal). Stuur een bericht, en je assistent gaat aan de slag.

Wat gebeurt er achter de schermen?

Dag 1: Server setup

We creëren een veilige virtuele server speciaal voor jou. Geïsoleerd van andere gebruikers.

Dag 1: Software installatie

OpenClaw wordt geïnstalleerd met alle benodigde componenten.

Dag 1-2: Configuratie

We koppelen je Telegram account, stellen de AI in, en configureren security.

Dag 2: Testen

We testen of alles werkt voordat we het aan jou overdragen.

Dag 2: Overdracht

Je krijgt toegang + een handleiding om te starten. Klaar om te ontdekken!

Wat heb jij nodig?

  • Een Telegram account (gratis)
  • Een Anthropic API key (we helpen je hiermee)
  • ~30 minuten voor de eerste kennismaking

En daarna?

Na de setup is de AI-assistent van jou. Je betaalt alleen voor:

  • Server hosting: ~€10-20/maand (direct bij de hostingpartij)
  • AI-tokens: ~€20-50/maand (direct bij Anthropic, afhankelijk van gebruik)

Geen abonnement bij ons. Geen verborgen kosten.
